# Neovim Dotfiles

Minha configuração do Neovim atual, baseada no video [Neovim Setup](https://www.youtube.com/watch?v=6pAG3BHurdM) do Josean Martinez.

Gerenciador de plugins: [lazy.nvim](https://github.com/folke/lazy.nvim)

## Plugins

[alpha-nvim](https://github.com/goolord/alpha-nvim) - Greeter
[nvim-autopairs](https://github.com/windwp/nvim-autopairs) - Fecha automaticamente múltiplos caracteres.
[colorscheme - catppuccin](https://github.com/catppuccin/nvim) - Tema do Neovim
[comment-nvim](https://github.com/numToStr/Comment.nvim) - Melhora os comentários
[dressing-nvim](https://github.com/stevearc/dressing.nvim) - Melhora elementos da UI padrão
[conform-nvim](https://github.com/stevearc/conform.nvim) - Formatter
[vim-tmux-navigator](https://github.com/christoomey/vim-tmux-navigator) - Mover entre buffers
[lazygit-nvim](https://github.com/kdheepak/lazygit.nvim) - Integra o LazyGit com o NeoVim
[nvim-lint](https://github.com/mfussenegger/nvim-lint) - Linting
[lualine-nvim](https://github.com/nvim-lualine/lualine.nvim) - Barra de status atualizada
[nvim-cmp](https://github.com/yanrdgs-dev/nvim-dotfiles/edit/main/README.md) - Autocomplete e Snippets
[nvim-tree](https://github.com/nvim-tree/nvim-tree.lua) - Árvore de arquivos
[substitute-nvim](https://github.com/gbprod/substitute.nvim) - Melhor substituição de código
[nvim-surround](https://github.com/kylechui/nvim-surround) - Adicionar pares de caractere em volta de código
[telescope-nvim](https://github.com/nvim-telescope/telescope.nvim) - Fuzzy Finder
[todo-comments-nvim](https://github.com/folke/todo-comments.nvim) - Comentários ToDo melhorados
[nvim-treesitter](https://github.com/nvim-treesitter/nvim-treesitter) - Highlighting
[trouble-nvim](https://github.com/folke/trouble.nvim) - Diagnóstico de erro e quickfix
[vim-maximizer](https://github.com/szw/vim-maximizer) - Maximiza o buffer atual
[which-key](https://github.com/folke/which-key.nvim) - Helper de keybinds
